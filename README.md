# DataBase-Azure-DIO
Neste desafio, pratiquei a criação de uma máquina virtual (VM) no Microsoft Azure, configurando sistema operacional, regras de acesso (como portas de RDP e SQL), e garantindo conectividade segura. A VM foi utilizada como ambiente de testes para conectar e gerenciar o banco de dados criado na nuvem.

Também realizei a criação de uma instância de banco de dados SQL no Azure, configurando parâmetros como nome do servidor, autenticação, camada de serviço e segurança (firewall). Com a instância ativa, testei a conexão via VM, utilizando o SQL Server Management Studio (SSMS), validando o funcionamento completo do ambiente. Esse processo foi essencial para entender na prática como serviços em nuvem podem ser integrados para criar uma infraestrutura funcional.
